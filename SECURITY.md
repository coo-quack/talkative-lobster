# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :white_check_mark: |
| < 1.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Talkative Lobster, please report it responsibly:

1. **Do not** open a public issue.
2. Email the maintainer or use [GitHub's private vulnerability reporting](https://github.com/coo-quack/talkative-lobster/security/advisories/new).
3. Include a description of the vulnerability, steps to reproduce, and the potential impact.
4. You can expect an initial response within 72 hours.
